Repositorio:
Un repositorio es el espacio donde Git almacena todos los archivos de un proyecto
junto con el historial completo de sus cambios. Puede estar en tu computadora
(repositorio local) o en un servidor en internet como GitHub (repositorio remoto).
 
Commit:
Un commit es una "fotografía" del estado de los archivos en un momento determinado.
Cada commit guarda los cambios realizados, quién los hizo y cuándo, lo que permite
volver a versiones anteriores del proyecto en cualquier momento.
 
Push:
El comando push sirve para enviar los commits realizados en el repositorio local
hacia el repositorio remoto (por ejemplo, GitHub). Es la forma de compartir y
publicar los cambios para que otros puedan verlos o colaborar en el proyecto.
